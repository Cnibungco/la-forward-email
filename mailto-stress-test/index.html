What to verify on each click

Opening the compose window is not the test. Comma BCC opens fine. The question is whether body text arrives intact.

  1. Paragraph breaks — blank lines between paragraphs, or one run-on block?
  2. Bullets — four separate • lines, or collapsed into one line / plain dashes / missing?
  3. Word spacing — any words smashed together? (Jason's Chrome bug)
  4. Subject — full line: "Support AB 123 — Tenant Protections Vote This Week" (spaces + em dash intact)
  5. BCC count — all 25 present? (secondary — check if you have time)
